- 소문자로만 이루어진 단어 S를 입력받는다.
- alphabet 배열로 a~z까지 0번~25번 인덱스로 나타낸다. 초기값은 -1
- 반복문을 이용하여 문자열을 탐색한다.
	- 문자 하나를 확인하여 문자열에서 해당 위치를 alphabet 배열에 대입한다.
	- for i=0 to length of S do:
		- position = S[i] - 'a'
		- if alphabet[position]이 -1 then:
			- alphabet[position] = i (알파벳 position에 문자열에서 문자의 위치를 대입)
- 문자열에 대한 탐색이 끝나면 공백으로 나누어 alphabet 배열을 출력한다.
