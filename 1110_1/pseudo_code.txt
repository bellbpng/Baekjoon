- 숫자 N을 입력받는다. 단 문자열의 형태로 입력받는다.
- 문자열의 형태로 입력받은 숫자를 정수로 변환하여 변수 N_number 에 대입한다.
- 졍수형 변수 temp는 0으로 초기화하여 시작, 사이클 횟수를 담는 cnt도 0으로 초기화한다.
- if N_number가 0 then:
	- 1 출력
	- 프로그램 종료
- while temp와 N_number가 다르면 do:
	- cnt 변수에 1을 더한다.
	- char형 변수 front, end 를 이용하여 숫자의 앞자리, 뒷자리 수를 담는다.
	- if length of N = 1 then front = '0', end = N[0]
	- else then front = N[0], end = N[1]
	- front와 end에 담겨 있는 문자형 숫자 데이터를 아스키코드값을 이용하여 정수값으로 변환한다.
	- 변환하여 각 자리값을 더한 결과를 K 변수에 대입한다.
	- 변수 temp에 N의 오른쪽 자리와 K의 오른쪽 자리를 이어붙여 정수값을 구한 후 대입한다.
- 반복문을 빠져나오면 cnt를 출력한다.
