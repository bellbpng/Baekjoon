- 단어의 개수 N 입력
- 각 줄에 단어를 하나씩 입력 받는다. 
- 그룹단어의 개수를 세는 변수 cnt를 선언하고 초기값은 0으로 한다.
- for i=0 to N-1 do:
	- 문자열 변수 str에 단어를 입력 받는다.
	- 벡터 컨테이너 v를 선언하여 연속된 문자를 하나의 문자로 표현한 새로운 문자열을 담는다.
	- for j=0 to length of str do:
		- v에 해당 문자를 push_back 한다.
		- while str[j] = str[j+1] do:
			- j 1증가 // 중복되는 문자는 인덱스를 지나치는 방법
	- v에 저장된 문자열의 각 문자 빈도수를 확인한다.
	- 빈도수가 2 이상이면 그룹단어가 아니다.
	- 그룹단어라면 변수 cnt를 1 증가시킨다.
- 반복문을 빠져나오면 cnt를 출력한다.
