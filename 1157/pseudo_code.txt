- 알파벳 대소문자로 된 단어를 입력받는다.
- 모든 문자를 대문자로 바꾼다.
	- toupper() 함수를 활용
- A~Z를 정적인 배열 alphabet에 0번~25번자리로 생각한다.
	- index = 해당문자 아스키코드값 - 'A'의 아스키코드값
- 정적인 배열의 초기값은 모두 0이다.
- 대문자로 변환된 문자와 매칭하는 알파벳을 발견하면 해당 값을 1 증가
	- 반복문을 통해 모든 문자에 대한 탐색을 진행
		- 대문자로 변환된 문자가 ch 변수라면 alphabet 배열에서 index = ch - 'A'
		- alphabet[index] 값을 1 증가
- alphabet 배열을 탐색하여 빈도수를 확인한다.
	- 모든 문자에 대한 탐색을 하여 max를 찾는다.
- 다시 한번 alphabet 배열을 탐색하여 max의 빈도수를 확인한다.
	- max의 빈도수와 같은 것이 2개 이상이 발견되면 ?를 출력하고 프로그램을 종료한다.
	- max가 빈도수가 1이라면 해당하는 문자를 출력하고 프로그램을 종료한다.
